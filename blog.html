What is the Document Object Model (DOM)?

DOM is Document Object Model which is a programming interface for web documents. DOM represents the structure of an HTML document as a tree of objects.

How do you select an element from the DOM?

To select an element from the DOM in JavaScript we can use several methods. Depending on the element such as id, class or tag name we can chose from the methods below:

  • If an element has an id attribute, we can use getElementById() to select it.
  • To select elements by their class name, we can use getElementsByClassName().
  • You can use getElementsByTagName() to select all elements with a specific tag name.
  • The most flexible and powerful method is to use querySelector() and querySelectorAll(), which let us select elements using CSS-style selectors.
What is event delegation in the context of the DOM, and why is it useful?

Event Delegation is a technique in JavaScript where you attach a single event listener to a parent element to handle events from its child elements. This is possible due to event bubbling, where events triggered on a child element "bubble up" to its ancestors.

How do you manipulate an element's attributes and styles using the DOM?

We can manipulate an element's attributes using the DOM in the following ways:

  • To set an attribute we can use setAttribute() method.
  • To get an attribute we can use getAttribute() method.
  • To remove an attribute we can use removeAttribute() method.

We can use style property in DOM to manipulate element's style. Three different cases of manipulating css style are:

  • We can directly assign the value of a style property of an element
  • We can usr multiple styles using cssText or classList method
  • The classList.toggle() method adds a class to an element if it doesn't exist, and removes it if it does, allowing you to easily switch styles on and off.
